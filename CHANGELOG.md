# Changelog

## 0.1.0 (2026-03-28)

### Features

- GitHub organization discovery with standard and deep scan profiles
- Multi-organization scanning via YAML config file
- JSON inventory, HTML report, and Excel workbook output
- GitHub App and Personal Access Token authentication
- GitHub Enterprise Server (GHES) support
- Capability-aware REST client with graceful degradation
- GraphQL bulk queries with budget tracking
- Workflow YAML parsing for deep Actions analysis
- Large file detection via recursive tree walk
- LFS detection from .gitattributes
- Security posture reporting (Dependabot, code scanning, secret scanning)
- Branch protection and ruleset discovery
- Organization member, package, and project inventory
- Bounded PostHog telemetry with error tracking
- PyInstaller standalone binary for Linux, macOS, Windows
- Offline HTML reports with SVG charts (no CDN dependencies)
- ASCII-only terminal output for CMD compatibility
- Cross-organization summary reports
